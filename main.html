The Advent of Code is an annual programming challenge held during the first 25 days in December. Each day at midnight eastern time, a new challenge is released. There are two challenges per day and the second challenge is unlocked by completing the first.
Each of my folders contains the following documents:
  • solution.py contains my coded solution for that day's challenge
  • problem.txt contains the challenges themselves
  • input.txt contains the input that I have to use in order to arrive at the solution

I do not explicitly provide the answers to these challenges to avoid spoilers. However, running my code will give you the answers to both parts of the challenge. My intent with sharing the code is to give an idea of my problem solving approach.
